## ✨ Features

### 🎨 Six Premium Resume Templates

| Template | Best For | Key Characteristics |
|---|---|---|
| **Creative** | Design, Tech, ISM roles | Sidebar layout, passport photo, Radar Chart, Emerald glassmorphism accents |
| **Minimalist** | Finance, Law, Management | Playfair Display typography, generous whitespace, clean layout |
| **Executive** | Management, Senior Roles | Dark headers, linear gradients (#0f172a to Accent), sidebar for expertise |
| **Luxe** | High-end Corporate, Luxury | Playfair Display large headings, letter-spaced roles, centralized contact info |
| **Custom** | Personalized Layouts | **Fully Dynamic Sidebars** - User selects side (Left/Right), width, and padding |
| **ATS-Friendly** | Online Portals / MNCs | Single-column, 0% images, text-only machine-readable (Arial) |

### 🛠 Custom Layout Designer (New!)
Exclusive to the **Custom Template**, users can now:
- **Switch Sidebar Side**: Toggle between Left or Right sidebars.
- **Adjust Width**: Set sidebar width (e.g., 25% or 40%) dynamically.
- **Layout Spacing**: Choose between Relaxed, Balanced, or Compact padding.

### 🧠 Smart Logic Guard
When **ATS Mode** is selected, the system automatically:
- Hides the passport photo
- Removes all Chart.js visualisations
- Switches to a plain single-column layout
- Ensures 100% machine-readability for Applicant Tracking Systems

### 🔄 Multi-Mode "Switch" Engine
- **Desktop**: Swaps the workspace view (Edit on Right vs Edit on Left).
- **Mobile**: Toggles between **Edit View** and **Live Preview** via a single floating button.

### ⚡ Live Theme Picker
Pick any Hex/Color value to instantly change the accent color for all templates, gradients, and icons.

### 📊 Radar Chart (Spider Chart)
Your Technical Skills automatically populate a **Radar / Spider Chart** via Chart.js, visualising multi-domain expertise at a glance.

### 💾 Auto-Save & Privacy
Progress is auto-saved to LocalStorage every 30 seconds. **No data ever leaves your device** — 100% private.

## 📋 Resume Data Modules

### 👤 Personal Information
- Passport photo upload with circular live preview.
- Name, Role, Phone, Email, Address, LinkedIn.
- **Professional Summary**: Now supports **Justified Text** for a cleaner look.

### 🎓 Education & 💼 Work Experience
- Dynamic addition/removal of cards.
- **Experience Descriptions**: Now support multiline text and text-justification.

### 🏆 University Activities & Achievements
- Integrated into **every template** for holistic profile building.

### ⚡ Skills Matrix
- Technical (Radar Chart + Progress Bars), Soft Skills (Body Pills), Tools (Body Pills).

### ✅ Extra Info / Benefits (Malaysian-Specific)
Visible across **all templates** now:
- **Licenses**: D, DA, B2, GDL, etc.
- **Languages**: BM, English, Mandarin, Tamil, Arabic.
- **Notice Period / Availability**: "Immediately", "1 Month", etc.
- **Expected Salary**: RM values.
- **Transport**: Own Transport toggle.

### 📋 References
- Integrated into sidbars or main bodies across all 6 templates.
